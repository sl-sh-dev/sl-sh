1//! Sl-console is a pure Rust, bindless library for low-level handling, manipulating
2//! and reading information about terminals.  This is a fork of termion.
3//!
4//! Sl-console aims to be simple and yet expressive. It is bindless, meaning that it
5//! is not a front-end to some other library (e.g., ncurses or termbox), but a
6//! standalone library directly talking to the TTY.
7//!
8//! Supports Mac OS X, Linux, and Windows (or, in general, ANSI terminals).
9//!
10//! For more information refer to the [README](https://github.com/sl-sh-dev/sl-console).
