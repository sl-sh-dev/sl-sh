Usage: (% int int)

Remainder from dividing (arg 1) by (arg 2). Note: Remainder and Modulo are two similar mathematical operations, called rem and rem-euclid. This function uses rem which is the same as the % operator in C. With rem, the sign of the result is the same as the dividend (arg 1). With rem-euclid, the sign of the result is always positive.
