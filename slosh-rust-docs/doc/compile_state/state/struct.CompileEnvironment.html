§use_line: bool§line: u32§specials: Option<Specials>§global_map: HashMap<Interned, usize>§gensym_idx: usize§namespace: Namespace§noop_map: HashMap<String, Value>

Some functions can be swapped to/from a noop at runtime for various reasons, e.g. make sure nothing can write to stdout; this field tracks the Value the function should point to so the user can swap that back in when they no longer want it to be a noop.
